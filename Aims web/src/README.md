
# AIMS Web – Internet Media Store

## Introduction
AIMS (Internet Media Store) is an e-commerce web application that allows customers to browse and purchase media products such as books, CDs, DVDs, and newspapers.  
The system also provides management features for administrators to manage products and customer orders.

## Features

### Customer
- Browse product list
- Search products by title or category
- Filter products by price range
- View product details
- Add products to cart
- Update or remove items in cart
- Place orders and provide delivery information
- Online payment via **VietQR** or **PayPal (Sandbox)**
- View and cancel orders via email link

### Product Manager / Admin
- Add new products
- Update product information
- Delete or deactivate products
- Manage product inventory
- Review and approve/reject customer orders
- Manage users and roles

## Technologies

### Frontend
- ReactJS

### Backend
- Node.js
- Express.js
- RESTful API
-  Spring Boot

### Database
- MongoDB
- Mongoose

### Payment Integration
- PayPal Sandbox API
- VietQR API
